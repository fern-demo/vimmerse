The Vimmerse Platform API lets you generate AI-powered videos from images, text prompts, or a combination of both. Transform static images into animated videos, create professional video stories, and build sophisticated video workflows — all through a simple REST API.

## What you can build

<CardGroup cols={2}>
  <Card title="Media" icon="video">
    Create animated videos from images or text prompts with support for multiple AI motion models, audio options, and customizable output formats.
  </Card>
  <Card title="Stories" icon="book-open">
    Generate complete video stories from text ideas or images. Use automatic mode for quick results or advanced mode for granular control over each scene.
  </Card>
  <Card title="Workflows" icon="diagram-project">
    Apply multi-step video processing pipelines using predefined templates for overlay effects, background replacement, face swapping, and more.
  </Card>
  <Card title="Images & Audio" icon="image">
    Generate and edit images with AI, create audio narration, background music, and sound effects to enhance your video content.
  </Card>
</CardGroup>

## Core capabilities

- **Image-to-Video** — Transform static images into animated videos with AI-generated motion
- **Text-to-Video** — Generate videos entirely from text descriptions
- **Story Generation** — Create multi-scene video stories with narration and background music
- **Video Workflows** — Apply complex processing pipelines with predefined templates
- **Multiple AI Models** — Choose from KlingAI, LumaAI, RunwayML, and more
- **Audio** — Add narration, sound effects, and background music to any video
- **Webhooks** — Receive real-time notifications when processing completes

## Base URL

All API requests are made to:

```
https://api.vimmerse.net
```
